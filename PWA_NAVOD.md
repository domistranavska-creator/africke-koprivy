# Africké kopřivy jako appka

## Co nahrát na hosting

Nahraj celý obsah složky `balicky/WEB_APP` na jeden webhosting, například Netlify, Vercel nebo GitHub Pages.

## Odkazy

- PC verze: `index.html`
- Mobilní verze: `mobile.html`

## Instalace v mobilu

1. Otevři odkaz na `mobile.html` v Chrome.
2. Klepni na menu se třemi tečkami.
3. Zvol `Přidat na plochu` nebo `Instalovat aplikaci`.
4. Přihlas se emailem, heslem a šifrovacím heslem.

## Instalace v PC

1. Otevři odkaz na `index.html` v Chrome nebo Edge.
2. Pokud se ukáže tlačítko `Instalovat appku`, klikni na něj.
3. Pokud se tlačítko neukáže, obnov stránku a v menu Chrome/Edge zvol `Instalovat aplikaci`.
4. Aplikace se přidá na plochu jako běžný program.

Poznámka: instalace se nabídne jen z HTTPS odkazu, například z GitHub Pages. Z lokálního souboru `file://` se jako PWA nenainstaluje.

## Sync a soukromí

Obsah se zobrazí až po přihlášení. Zákazníci a objednávky jsou v Supabase uložené zašifrovaně. Fotky jsou v privátním úložišti a syncují se v plné kvalitě.
